# piksel

**piksel** is a lightweight C++ graphics library built around **OpenGL**, **GLM**, and **GLFW**.  
It provides a minimalist, clean interface for rendering, math operations, and window/context handling.

## Features

- Modern OpenGL rendering
- GLFW-managed windows and input
- GLM-based math (vectors, matrices, transforms)
- Cross-platform (Windows, Linux, macOS)

## Requirements

- C++20 or newer  
- OpenGL  
- [GLFW](https://www.glfw.org/)  
- [GLM](https://github.com/g-truc/glm)
- [stb](https://github.com/nothings/stb)
- [glad](https://github.com/Dav1dde/glad)
- [imgui](https://github.com/ocornut/imgui)
